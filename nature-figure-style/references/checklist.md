# 出图后的检查清单

每画完一张 Nature 风格图，**在报告完成之前**必须照这清单走一遍。漏一项就有破坏 Nature 感的小毛病。

## 通用 (任何图都要查)

- [ ] **字体是 sans-serif**，不是 Times / serif
- [ ] **panel 标号**：粗体 / 小写 / a-b-c 形式（不是 A、不是 (a)、不是 1.2.3）
- [ ] **panel 标号位置**贴左上角，与子图边对齐
- [ ] **配色**来自 colormap-palette，不是 matplotlib 默认 tab10 / viridis
- [ ] **dpi ≥ 300**（PNG）或矢量格式（PDF / EPS）
- [ ] **画布尺寸**符合栏宽：单栏 3.5 in / 1.5 栏 4.7 in / 双栏 7.2 in
- [ ] **保存时 `bbox_inches='tight'`**，没有多余白边
- [ ] **没有 matplotlib 报警告**（`UserWarning: findfont` 等）

## 数据图额外检查

- [ ] **轴标签格式**：`量名 (单位)`，圆括号不是方括号
- [ ] **`(a.u.)` 用于无量纲量**，不写 `(arb. units)` / `(normalized)`
- [ ] **spine**：默认四面闭合（谱图、log-log、polar、密集时序、heatmap）；简洁折线/柱状用 `half_frame(ax)` 切 L+B；选好后整张图统一，不要一图两风
- [ ] **tick 方向是 inward（in）**——这是 Nature 最稳定 fingerprint，朝外立马显业余
- [ ] **闭合方框时 tick 镜像到 top/right**（`ax.tick_params(top=True, right=True)`）
- [ ] **有 minor ticks**（log 轴必有）
- [ ] **major tick 数量 ≤ 6**（不要挤一堆数字）
- [ ] **legend 无边框**（`frameon=False`）
- [ ] **legend 字号 6-7 pt**
- [ ] **数据线宽 0.8-1.2 pt**，没有粗到 2 pt+
- [ ] **轴线宽 ≈ 0.6 pt**
- [ ] **没有打开默认 grid**（如要用 grid 必须 alpha < 0.3）
- [ ] **数据点 marker 大小 3-5 pt**，不要超过 6 pt
- [ ] **误差棒 elinewidth ≤ 0.8 pt，capsize ≤ 2**；优先用 shaded band（alpha 0.15-0.25）
- [ ] **colorbar fraction 约 0.04, pad 约 0.02**
- [ ] **colorbar 的 outline linewidth 0.5 pt**
- [ ] **双 y 轴**：左右轴颜色 + 刻度 + 数据线 颜色一致
- [ ] **inset 子图边框 0.4 pt**

## 架构图额外检查

- [ ] **`set_aspect('equal')`** 已设置
- [ ] **`axis('off')`** 已设置
- [ ] **元件圆角统一**（`rounding_size` 全图相同）
- [ ] **元件边框线宽统一**（0.5-0.8 pt）
- [ ] **箭头线宽 ≥ 元件边框线宽**
- [ ] **路径线宽统一**（光路 1.2-1.4 pt）
- [ ] **辅助/反馈路径用虚线**（dashes=(3,2) 或 dashes=(4,2)）
- [ ] **元件标注字号 ≥ 6.5 pt**
- [ ] **元件之间留白 ≥ 1 倍元件高度**
- [ ] **缩写符号**（如 EOM、PD、AWG）首次出现时给全称
- [ ] **信号流向**：从左到右、从上到下
- [ ] **3D 阴影 alpha < 0.2**（如有）
- [ ] **没有任意角度斜线**（除非物理上必要）

## 颜色一致性检查

- [ ] **同一变量在多子图中使用同一颜色**（"sample A"在 a/b/c 子图都是同一蓝色）
- [ ] **legend 颜色 = 数据线颜色**（matplotlib 99% 时候自动对，但 twin axes 容易错）
- [ ] **colorbar 范围合理**（不是 0-1 但数据 0.001-0.01 这种）
- [ ] **配色对色盲友好**（用 simulator 看，避开纯红 + 纯绿对比）

## 文字内容检查

- [ ] **没有 caption 性的长句子作为标题**——标题应该简短或留空
- [ ] **轴标签首字母大写**（"Wavelength" 不是 "wavelength"）
- [ ] **单位用罗马体**（"$\\mathrm{Wavelength}$ (nm)"），变量用斜体（"$\\lambda$ (nm)"）
- [ ] **希腊字母用 LaTeX**：`r'$\\lambda$'` 而不是 `'λ'`（除非确认字体支持）
- [ ] **下标 / 上标 用 LaTeX**：`r'$f_\\mathrm{rep}$'` 而不是 `'f_rep'`
- [ ] **角度单位 `°`**，不是 `deg`

## 印刷可读性检查（终极测试）

把出图缩到 50% 看一遍——如果在 50% 大小下：
- [ ] 刻度数字仍能看清
- [ ] 轴标签仍能看清
- [ ] panel 标号仍清晰
- [ ] 数据线还能区分

如果哪一项糊了，**字号需要加大**或者**画布需要缩小**（让相对字号变大）。

## 终审：让 Nature 编辑视角看一眼

模拟 reviewer 视角，问自己：
1. **第一眼**能不能 5 秒内看出这张图想说什么？
2. **配色**会不会显得"年轻"或"业余"（饱和度过高、颜色太多）？
3. **细节**有没有破坏严肃感（grid 太显眼、阴影太重、字体不一致）？
4. **如果 reviewer 要复印这张图**，会不会因为颜色太淡而看不清？

任何一项犹豫，回去调。
